Sistema de Nomina

Este proyecto es una aplicación de consola en C# de un sistema de gestión de pagos de la nomina de los empleados tomando en cuenta el documento de requerimiento.
Permite registrar diferentes tipos de empleados, ver la lista de empleados, actualizar información y generar reportes semanales.
